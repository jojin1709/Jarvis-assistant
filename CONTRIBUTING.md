# Contributing to JX Jarvis

JX Jarvis is a source-available proprietary project created by Jojin John.

Public contributions are welcome for review, but contributors do not receive
direct edit access to the official repository. All changes must be submitted
through pull requests and must be approved by Jojin John before becoming part of
the official project.

## Rules

- Do not commit API keys, secrets, tokens, `.env` files, memory data, logs, or
  private user files.
- Do not remove license, ownership, creator, security, or attribution notices.
- Do not publish modified builds as official JX Jarvis releases.
- Do not use the JX Jarvis name, logo, or branding for another app without
  written permission.
- Keep changes focused and production-ready.
- Preserve permission checks for desktop, browser, terminal, and file actions.
- Never fake task execution in the UI.

## Pull Requests

Pull requests are suggestions only until accepted by Jojin John. By opening a
pull request, you grant Jojin John permission to use, modify, and distribute the
submitted contribution as part of JX Jarvis.

## Official Releases

Only Jojin John can publish official JX Jarvis releases.
